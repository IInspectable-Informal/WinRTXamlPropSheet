# WinRTXamlPropSheet

## 简介
写来玩玩的玩具项目  
它仅证明了 XAML Island 可以嵌入到 explorer 属性页  
由于存在大量已知/未知问题，该项目仅供体验，请勿用于生产环境！！！  

## 接手必备能力
### 核心语言技能
| 技能点   | 具体要求                                                           |
|:---------|:-------------------------------------------------------------------|
| C++ 基础 | 熟悉现代 C++（C++17 及以上），掌握 RAII、智能指针等现代特性            |
| 内存管理 | 理解裸指针操作、手动堆内存分配与释放（new / delete）                   |
| 标准库   | 熟练使用 STL 容器（std::map、std::vector 等）和同步机制（std::mutex） |

### Windows 平台开发
| 技能点         | 具体要求                                                      |
|:---------------|:--------------------------------------------------------------|
| Win32 API      | 熟悉窗口管理、消息循环等核心 API                                |
| Win32 GUI 开发 | 掌握窗口类注册、窗口过程（WndProc）、对话框编程                   |
| COM 编程       | 理解 COM 架构、接口、类工厂、DLL 注册与调用机制                   |
| Shell 扩展     | 熟悉 IShellExtInit、IShellPropSheetExt 接口及与 explorer 的集成 |
